WINDOWS
Запустите от имени администратора нужный файл:
- discord.bat - для обхода Discord.
- discord_youtube.bat - для обхода Discord и YouTube.
- service_discord.bat - для обхода Discord с автозапуском.
- service_discord_youtube.bat - для обхода Discord и YouTube с автозапуском.
- service_goodbye_discord.bat - для использования с сервисом GoodbyeDPI, чтобы обходить только Discord. Запускайте после создания сервиса GoodbyeDPI. Если сервис вылетит, перезапустите устройство.
- service_remove.bat - для остановки и удаления вышеуказанных сервисов.

РЕШЕНИЕ ПРОБЛЕМ
Убедитесь, что файлы запускаются от имени администратора.
Если bat-файлы не работают, выполните очистку от драйвера WinDivert (инструкция ниже)
Если сервис не работает, проверьте, чтобы в пути к файлу не было пробелов и русских символов. Отключите программы, мешающие созданию сервиса (антивирусы, клинеры).
Если не работает с VPN, отключите функцию TUN (Tunneling).
Если service_goodbye_discord не работает, убедитесь, что сервис GoodbyeDPI запущен и называется GoodbyeDPI. Перезапустите устройство после запуска service_goodbye_discord.bat.
Если видео на YouTube не загружается, попробуйте установить Kyber и QUIC в default через chrome://flags/. Также измените строку с --filter-tcp=443, заменив --dpi-desync-fooling=md5sig на --dpi-desync-fooling=badseq.
Попробуйте обновить бинарники из оригинального репозитория.
Для проблем с YouTube или Discord ищите ответы в соответствующих Issues на GitHub.

ОСТАНОВКА И УДАЛЕНИЕ ОБХОДА (WinDivert)
Для удаления драйвера WinDivert откройте командную строку от имени администратора и выполните следующие команды:
sc stop WinDivert
sc delete WinDivert

Если драйвер записан по-другому, уточните его название командой: driverquery | find "Divert".

ДОБАВЛЕНИЕ ДОПОЛНИТЕЛЬНЫХ АДРЕСОВ И ЗАБЛОКИРОВАННЫХ САЙТОВ
Список можно дополнить, используя файлы list-general.txt (для discord_youtube) и list-discord (для файлов без youtube в названии).
После добавления сервис нужно перезапустить.
